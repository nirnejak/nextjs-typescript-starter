## Available Scripts

**Install Dependencies**

```bash
npm install
```

**Setup Pre-commit**

```bash
npm run prepare
```

**Start Development Server**

```bash
npm run dev
```

**Build for Production**

```bash
npm run build
```

**Start Production Server**

```bash
npm start
```
